# calendar.py
## скрипт делает налоговый календарь на следующий год из присланного эталона

Использование:
* активировать виртуальную среду venv
> call C:\projects\py\calendar\venv\Scripts\activate.bat
* добавить в cmd сценарий следующие строки:
> C:\projects\py\calendar\main.py -nsr действующий_нк_субъекта.nsr -etalon etalon.nsr -topic новый_топик

расшифровка ключей:
* -nsr [файл с действующим нк субъекта.nsr] - действующий налоговый календарь региона
* -etalon [etalon.nsr] - новый эталонный нср налогового календаря 
* -topic [НОВЫЙ ТОПИК] - свободный номер топика

В результате измененные топики можно будет импортировать из папки result
